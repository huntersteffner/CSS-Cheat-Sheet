CSS Syntax Cheet Sheet

Here is a little cheat sheet of a few CSS Tricks. Go ahead and click on one and see what happens.

Transform

            .transform {
                transform: translateX(2rem)
            }

Flex Row

            .row {
                display: flex;
                flex-direction: row;
            }

Tool Tip

            span:hover::before {
                position: absolute;
                transform: translate(1rem, -1rem);
                padding: .5rem .7rem;
                background-color: #13293d;
                color: #fff;
                border-radius: 5px;
              }

Pseudo Class

            .pseudo:hover {
                background-color: #fff;
            }

Psuedo Element

            .pseudo::after {
                color: #fff;
                content: 'These are words.'
            }

Rotate

            .rotate {
                transform: rotate(15deg)
            }
